# Examples

このディレクトリは、`/docs` 運用時の記入例をまとめたサンプルです。

## 収録内容

- `PROJECT.example.md`
  - `PROJECT.md` の良い記入例
- `IMPLEMENTATION-RECORD.example.md`
  - 1タスク分の設計・実装・レビュー記録例

## 使い方

1. `PROJECT.example.md` を参考に、実プロジェクトの `PROJECT.md` を記入する
2. 実装タスクごとに `IMPLEMENTATION-RECORD.example.md` の形式で記録する
3. `quality/definition-of-done.md` と `super-mode/final-gate.md` で完了判定する
